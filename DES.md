# 整体介绍
1. config.json 系统配置接口
2. infowindow.json 首页信息窗口
3. productlist.json 产品列表
4. imgs 图片目录-子目录推送使用bundle id
5. msg 是消息目录 msg.json是消息内容
6. doc 是产品说明文件目录
7. detail目录是产品详情目录，bundle id 作为json的名称，方便调用 logs目录是更新内容日志
